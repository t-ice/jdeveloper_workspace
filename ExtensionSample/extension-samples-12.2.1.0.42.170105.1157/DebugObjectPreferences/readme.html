JDeveloper Extension SDK Sample: DebugObjectPreferences

This sample demonstrates extension manifest support for registering custom object preferences for the debugger. It installs a custom object preference that displays the hue of a color when it is shown in debugger windows.

To see this extension in action, deploy it to JDeveloper or any other FCP application and then follow these steps:

  1. Create a new project.
  2. Create a new Java file with the following code: 
    package mypackage;

import java.awt.Color;

public class Class1
{
  public static void main( String[] args )
  {
    Color red = Color.RED;
    Color green = Color.GREEN;
  }
}
  3. Create a breakpoint on line 10 (green = Color.GREEN;)
  4. Start debugging the project.
  5. When you hit the breakpoint, notice that the hue of red is shown in the debugger's Data window.

For more information about this sample, see the comments in extension.xml.
