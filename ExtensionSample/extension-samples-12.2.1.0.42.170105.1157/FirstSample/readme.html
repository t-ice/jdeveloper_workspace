Oracle Fusion Client Platform Extension SDK Samples

FirstSample Sample

Overview

This sample extension installs the following functionality into the product:

  • A gallery item visible via File->New...
  • An item in the File menu.
  • A toolbar button
  • An item in the context menu of the navigator and editor.

Contents

File Description
extension.xml The extension manifest. This declares all the functionality provided by the extension.
SimpleContextMenuListener.java A context menu listener, responsible for adding a menu item for an action declared in extension.xml to the navigator and editor context menus.
SimpleController.java The controller determines when the action is enabled, and does the actual work when the user invokes the action. In this sample, the action delegates to the gallery item wizard.
SimpleWizard.java A description of a gallery item wizard. The invoke() method of this class brings up an alert.
